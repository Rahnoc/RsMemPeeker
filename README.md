RS Memory Peeker

Agent application.

根據記憶體壓力更改選單列 icon 的顏色 (綠/橘/紅)。

顯示 swap 用量並以顏色提示， 未使用灰色。 5G 之前橘色， 5G 之後紅色。 (8G用戶可改為 2G 就跳色。)

額外可以顯示硬碟剩餘空間。

提供按鈕直接開啟 "活動監視器" 或 "系統資訊 > 一般 > 儲存空間"。

### * 沒有觀測之外的機能

-----

2024 Jun，入手 Macbook Air M3 8G 256G。 預裝 Sonoma。

Xcode 15.4 (15F31d) 時期，iOS_17.5_Simulator_Runtime 還能秒開。 iOS 18 開始出現變慢問題。

iOS 26 simulator 之類開始啟動時更是嚴重，會開始吃 swap 到 8GB ~ 12GB 後，才彈出模擬器視窗。

雖然執行期間的速度正常，但是等待開啟時的 swap 時間太久。 而 SwiftUI 的 canvas 預覽因為也是在用模擬器，所以同樣要等 swap。

加上 Xcode 在這幾版開始要求需裝對應 Xcode 版本的新模擬器後，才能 build iOS 專案。 不能像以前隨便裝個iOS 15.3 就開始編譯，一定要卡個最新版，不論有沒有用到都要下載。

於是 專案 run 時有記憶體壓力，開發時更新 Xcode 有下載 ios runtime simulator 的 SSD 容量消耗。

每次都要手動開 活動監視器 太麻煩，於是本 RsMemPeeker 提供記憶體壓力與硬碟容量資訊，以及按鈕連結，可直接開啟 *活動監視器* 與 系統資訊內的 *儲存空間* 硬碟用量。

苗頭不對時可以關閉 Safari 或 Chrome 等不用的東西來緩和記憶體壓力。 或是把檔案移到外接硬碟空出 SSD 來**認真 swap**。
